Ⅰ. 의사결정트리 개요
의사결정트리는 데이터의 특성(Feature)을 기준으로 노드를 분할하여 트리 형태의 분류·회귀 모델을 구성하는 지도 학습 알고리즘이다. 어떤 특성으로, 어떤 값에서 분할할지를 결정하는 기준이 핵심이다.
Ⅱ. 정보이득 (Information Gain) — ID3, C4.5
엔트로피 H(S) = -Σ pᵢ · log₂(pᵢ) (pᵢ: 클래스 i의 비율) 정보이득 IG(S, A) = H(S) - Σ (|Sᵥ|/|S|) · H(Sᵥ) (A: 특성, Sᵥ: A=v인 부분집합) 예시: 14개 데이터 중 긍정 9개, 부정 5개 H(S) = -(9/14)log₂(9/14) - (5/14)log₂(5/14) ≈ 0.940 bits 속성 A 분할 후: H(S₁)=0.811, H(S₂)=1.000 IG(S,A) = 0.940 - [(8/14)×0.811 + (6/14)×1.000] ≈ 0.048 bits
특징
• 값의 종류가 많은 특성(고유도 높은 속성) 선호 편향 발생 → 이득비로 보완
• ID3 알고리즘에서 사용, 다중 분기 트리 생성
Ⅲ. 이득비 (Gain Ratio) — C4.5
분할정보 SplitInfo(S, A) = -Σ (|Sᵥ|/|S|) · log₂(|Sᵥ|/|S|) 이득비 GR(S, A) = IG(S, A) / SplitInfo(S, A) 목적: 값의 종류가 많은 특성에 불이익(패널티) 부여 → 정보이득의 고유도 편향 보정
특징
• C4.5 알고리즘에서 사용. 연속형 특성 처리 가능 (이진 분할)
• 결측값 처리, 가지치기(pruning) 지원. ID3보다 실용적
Ⅳ. 지니 불순도 (Gini Impurity) — CART
Gini(S) = 1 - Σ pᵢ² (pᵢ: 클래스 i의 비율) 분할 후 가중 지니: Gini(S, A) = Σ (|Sᵥ|/|S|) · Gini(Sᵥ) 지니 감소량: ΔGini = Gini(S) - Gini(S, A) → 최대화하는 A 선택 예시: 긍정 3/5, 부정 2/5인 노드 Gini = 1 - (3/5)² - (2/5)² = 1 - 0.36 - 0.16 = 0.480
특징
• CART(Classification and Regression Trees) 알고리즘 사용
• 항상 이진 분할(Binary Split). 연속형·범주형 모두 처리
• 로그 연산 불필요 → 계산 속도가 엔트로피보다 빠름
• sklearn의 DecisionTreeClassifier 기본값
Ⅴ. 3가지 분할 기준 비교
기준수식범위사용 알고리즘특징
정보이득IG = H(S) - 가중엔트로피0 이상ID3고유도 편향, 다중 분기
이득비GR = IG / SplitInfo0~1C4.5, C5.0편향 보정, 결측 처리
지니 불순도Gini = 1 - Σpᵢ²0~0.5(이진)CART빠른 계산, 이진 분할
Ⅵ. 과적합 방지 — 가지치기(Pruning)
사전 가지치기 (Pre-pruning)

트리 성장 중 최소 샘플 수(min_samples_split), 최대 깊이(max_depth), 최소 불순도 개선량 임계값 설정으로 성장 조기 중단

사후 가지치기 (Post-pruning)

결론 — 감리·실무 포인트

Ⅰ. K-Means 개요 및 동작 원리
K-Means는 데이터를 K개의 군집으로 분할하는 비지도 학습 군집화 알고리즘으로, 각 데이터 포인트를 가장 가까운 중심점(Centroid)에 할당하고 중심점을 반복 갱신하여 수렴하는 방식이다.
K-Means 알고리즘 동작 절차 1단계 [초기화] K개의 초기 중심점(centroid) μ₁,...,μₖ 임의 선택 2단계 [할당] 각 데이터 xᵢ를 가장 가까운 중심점의 군집에 할당 cᵢ = argmin_k ||xᵢ - μₖ||² 3단계 [갱신] 각 군집의 중심점을 소속 데이터의 평균으로 갱신 μₖ = (1/|Cₖ|) Σ xᵢ (xᵢ ∈ Cₖ) 4단계 [반복] 2~3단계를 중심점 변화가 없을 때까지(또는 ε 이하) 반복 수렴 보장: 유한 번의 반복으로 반드시 수렴 (단, 지역 최솟값 가능)
Ⅱ. 목적함수 (WCSS)
목적함수 WCSS (Within-Cluster Sum of Squares, 군집 내 제곱합) J = Σₖ Σ_{xᵢ∈Cₖ} ||xᵢ - μₖ||² → 최소화 목표 WCSS가 작을수록 군집 내 응집도가 높음 (밀집된 군집) 단, K를 늘릴수록 WCSS는 단조 감소 → 최적 K 판단 기준 필요
Ⅲ. 초기 중심점 문제 및 K-Means++ 개선
⚠️ 초기 중심점 선택의 문제
임의 초기화 시 지역 최솟값(Local Optimum)에 빠질 수 있어 군집 품질이 불안정. 실행마다 결과가 다를 수 있음
✅ K-Means++ (Arthur & Vassilvitskii, 2007) 개선 방법
1. 첫 번째 중심점 μ₁을 데이터 중 균등 무작위 선택
2. 나머지 중심점 μₖ를 기존 중심점과의 거리²에 비례한 확률로 선택
→ 먼 곳의 데이터가 다음 중심점으로 선택될 확률 높음
3. K개 중심점 확보 후 일반 K-Means 실행
효과: WCSS O(log K) 근사 보장, 수렴 속도 향상, 안정적 결과
Ⅳ. 최적 K 결정 방법
📐 Elbow Method (엘보우 방법)

K=1,2,...,n에 따른 WCSS 변화를 시각화. WCSS가 급격히 감소하다 완만해지는 '팔꿈치(Elbow)' 지점을 최적 K로 선택. 주관적 판단 필요

📐 실루엣 계수 (Silhouette Coefficient)

s(i) = (b(i) - a(i)) / max(a(i), b(i))
a(i): 같은 군집 내 평균 거리 (응집도)
b(i): 가장 가까운 타 군집까지 평균 거리 (분리도)
범위: -1~+1. +1에 가까울수록 우수. 평균 실루엣이 최대인 K 선택

Ⅴ. K-Means 한계 및 대안 알고리즘
한계원인대안 알고리즘
K 사전 지정 필요K가 미지인 경우 적용 어려움DBSCAN, 계층적 군집화
구형 군집만 탐지유클리드 거리 기반Spectral Clustering, DBSCAN
이상값(Outlier) 민감평균 기반 중심점K-Medoids (PAM)
스케일 민감거리 기반 계산사전 표준화(StandardScaler) 필수
결론

K-Means는 대용량 데이터에 효율적인 군집화 알고리즘으로, 고객 세분화·문서 군집화·이미지 압축 등에 널리 활용된다. 공공 데이터 분석 시 사전 표준화, K-Means++ 초기화, 실루엣 계수를 통한 K 결정의 세 단계를 거쳐 신뢰성 있는 군집 결과를 도출해야 한다.

Ⅰ. 연관 규칙 마이닝 개요
연관 규칙 마이닝은 대규모 트랜잭션 데이터에서 항목 간의 흥미로운 상관 관계를 발견하는 기법이다. "X를 구매한 고객은 Y도 구매한다(X→Y)"와 같은 패턴을 탐지하며, 장바구니 분석·추천 시스템·크로스셀링 전략에 활용된다.
Ⅱ. 3가지 평가 척도
지지도 (Support): 전체 트랜잭션 중 X와 Y가 함께 등장하는 비율 Support(X→Y) = P(X∪Y) = |T(X∪Y)| / |T| 신뢰도 (Confidence): X가 등장한 트랜잭션 중 Y도 등장하는 비율 [조건부 확률] Confidence(X→Y) = P(Y|X) = Support(X∪Y) / Support(X) 향상도 (Lift): X 구매가 Y 구매에 얼마나 영향을 주는지 (독립성 대비) Lift(X→Y) = Confidence(X→Y) / Support(Y) = P(X∪Y) / (P(X)·P(Y)) 해석: Lift>1 양의 상관, Lift=1 독립, Lift<1 음의 상관 예시: T=100건, X∪Y=30건, X=40건, Y=50건 Support = 30/100 = 0.30 (30%) Confidence = 0.30/0.40 = 0.75 (75%) Lift = 0.75/0.50 = 1.5 → 양의 연관성
Ⅲ. Apriori 알고리즘 동작 원리
Apriori 원리 (핵심 성질)
빈번한 항목집합의 모든 부분집합도 빈번하다 → 역도: 비빈번한 항목집합의 상위 집합은 비빈번
→ 최소 지지도 미충족 집합 발견 시 해당 집합을 포함하는 모든 상위 집합을 가지치기 (Pruning)
Apriori 동작 절차 (min_support = 0.3 가정) 1단계: 모든 1-항목집합의 지지도 계산 → min_support 이상인 L₁ 생성 2단계: L₁로부터 2-항목집합 후보(C₂) 생성 → 지지도 계산 → L₂ 생성 3단계: L₂로부터 C₃ 생성 → 가지치기(부분집합이 L₂에 없으면 제거) → L₃ 생성 ...반복... 더 이상 빈번 항목집합이 생성되지 않을 때까지 시간복잡도: O(2^n · |T|) — 항목 수 n에 지수적 증가 문제점: 데이터베이스를 반복 스캔 (k-항목집합마다 1회 스캔)
Ⅳ. Apriori 한계 및 FP-Growth 개선
Apriori의 한계
① 다중 데이터베이스 스캔: k 단계마다 전체 DB 스캔 → I/O 비용 높음
② 방대한 후보 집합 생성: 항목 수 n이 클 때 후보 수 폭증 (2ⁿ-1)
③ 메모리 사용: 대용량 후보 집합을 메모리에 유지
✅ FP-Growth (Frequent Pattern Growth) 개선
핵심 아이디어: 데이터베이스를 FP-Tree(압축 트리 구조)로 2회만 스캔하여 후보 생성 없이 빈번 패턴 추출

동작 절차:
1. DB 1차 스캔 → 빈번 1-항목집합 추출 및 빈도 내림차순 정렬
2. DB 2차 스캔 → 트랜잭션별 정렬된 항목으로 FP-Tree 구성
3. FP-Tree에서 조건부 패턴 베이스(Conditional Pattern Base) 추출
4. 조건부 FP-Tree로부터 재귀적 빈번 패턴 마이닝

성능 비교: Apriori 대비 10~100배 빠른 수행 속도, 메모리 효율적
결론

연관 규칙 마이닝은 전자상거래 추천 시스템, 의료 진단 패턴 발견, 공공 민원 서비스 패턴 분석 등에 활용된다. 대용량 데이터 환경에서는 FP-Growth 또는 분산처리 기반 PFP-Growth(Spark)를 적용하며, 향상도(Lift)가 1.2 이상인 규칙을 실용적 연관 규칙으로 선별하는 것이 일반적이다.

Ⅰ. 개요 — SVM의 개념
SVM(Support Vector Machine)은 Vapnik이 1995년 제안한 지도 학습 알고리즘으로, 두 클래스 사이의 결정 경계(초평면)를 최대 마진으로 설정하여 일반화 성능을 극대화하는 분류·회귀 모델이다. 구조적 위험 최소화(SRM) 원리에 기반하며, 소규모 고차원 데이터에서 탁월한 성능을 발휘한다.
Ⅱ. 가. 최대 마진 분류 원리
결정 초평면: w·x + b = 0 (w: 가중치 벡터, b: 편향) 마진(Margin) = 2 / ||w|| → 마진 최대화 = ||w|| 최소화 최적화 문제 (하드 마진): 최소화: (1/2)||w||² 조건: yᵢ(w·xᵢ + b) ≥ 1 (∀i) 서포트 벡터(Support Vector): 결정 경계에 가장 가까운 데이터 포인트 → yᵢ(w·xᵢ + b) = 1인 샘플들. 초평면 결정에만 영향

하드 마진(Hard Margin): 모든 샘플이 마진 경계 밖에 위치해야 함. 선형 분리 가능 데이터에만 적용 가능. 이상값(Outlier)에 민감

소프트 마진(Soft Margin): 슬랙 변수(ξᵢ≥0) 도입으로 일부 오분류 허용
최소화: (1/2)||w||² + C·Σξᵢ (C: 정규화 파라미터)
C가 크면: 오분류 허용 적음(하드 마진에 가까움, 과적합 위험)
C가 작으면: 오분류 허용 많음(더 넓은 마진, 과소적합 위험)
Ⅲ. 나. 커널 트릭 (Kernel Trick)
커널 트릭은 데이터를 명시적으로 고차원 공간에 투영(φ(x))하지 않고, 커널 함수 K(xᵢ,xⱼ)=φ(xᵢ)·φ(xⱼ)를 통해 내적(dot product)만 계산하여 비선형 분류를 가능하게 하는 기법이다.
선형 분류 불가 → 고차원 특성 공간으로 매핑 후 선형 분류 가능 x → φ(x) : 저차원 → 고차원(또는 무한 차원) 매핑 커널함수: K(xᵢ,xⱼ) = φ(xᵢ)ᵀ·φ(xⱼ) (내적 계산만으로 효율적 처리)
커널 함수수식특징적합 데이터
선형 (Linear)K(x,z) = xᵀz커널 트릭 미사용. 고차원 희소 데이터에 효율적텍스트 분류, 유전자 데이터
다항식 (Polynomial)K(x,z) = (γxᵀz+r)ᵈd차 다항 특성 생성. 차수 d 선택 필요이미지 분류, 자연어처리
RBF/가우시안K(x,z) = exp(-γ||x-z||²)무한 차원 매핑. 가장 널리 사용. γ로 결정경계 유연성 조절대부분의 분류 문제 (기본값)
시그모이드K(x,z) = tanh(γxᵀz+r)신경망과 유사. 양정치(PSD) 보장 안 됨신경망 대안으로 제한적 사용

C와 γ 하이퍼파라미터 튜닝 전략

γ (RBF 커널 폭): 크면 결정 경계 복잡·지역적(과적합), 작으면 부드러움(과소적합)

Ⅱ. 가. MLP 구조 및 역전파 학습 원리
MLP 구조: 입력층(x₁,...,xₙ) → [은닉층1: h=σ(Wx+b)] → [은닉층2: ...] → 출력층(ŷ) 순전파 (Forward Propagation): zˡ = Wˡ·aˡ⁻¹ + bˡ (선형 변환) aˡ = σ(zˡ) (비선형 활성화 함수 적용) 최종: ŷ = softmax(z^L) (분류) 또는 선형(회귀) 손실함수: L = -(1/n)Σ[y·log(ŷ) + (1-y)·log(1-ŷ)] (교차 엔트로피) 역전파 (Backpropagation) — 연쇄 법칙(Chain Rule): δL/δWˡ = δL/δaˡ · δaˡ/δzˡ · δzˡ/δWˡ Wˡ ← Wˡ - η·(δL/δWˡ) (경사 하강법, η: 학습률)

Ⅱ. 나. PCA (주성분 분석) 원리 및 동작 절차
PCA는 데이터의 분산을 최대로 보존하는 새로운 직교 좌표계(주성분)를 찾아 고차원 데이터를 저차원으로 선형 변환하는 비지도 차원 축소 기법이다.
PCA 수학적 원리: 1. 데이터 중심화: X_centered = X - μ (평균 차감) 2. 공분산 행렬 계산: C = (1/n) X_centeredᵀ · X_centered (p×p 행렬) 3. 고유값 분해(Eigendecomposition): C = V · Λ · Vᵀ V: 고유벡터 행렬 (주성분 방향), Λ: 고유값 대각행렬 (분산량) 4. 고유값 내림차순 정렬 → 상위 k개 고유벡터 선택 (주성분) 5. 저차원 투영: Z = X_centered · Vₖ (n×k 행렬) 설명 분산 비율: Σᵢ₌₁ᵏ λᵢ / Σᵢ₌₁ᵖ λᵢ → 누적 분산 95% 이상이 되는 k 선택

(Autoencoder)		신경망으로 인코더(압축)·디코더(복원) 학습. 병목층이 저차원 표현고도 비선형. 대규모 데이터 적합복잡한 비선형 구조 학습학습 비용 높음, 해석 어려움이미지·음성 압축, 이상 탐지
차원 축소 기법 선택 가이드 목적: 전처리(특성 추출) → PCA (빠름, 결정론적, 역변환 가능) 목적: 2D/3D 시각화 → t-SNE (군집 구분 우수) 또는 UMAP (빠름, 대용량) 목적: 딥러닝 특성 표현 학습 → Autoencoder (VAE 포함) 데이터: 선형 구조 강함 → PCA 데이터: 비선형·다양체 구조 → t-SNE / UMAP / Kernel PCA
Ⅳ. 결론

차원 축소는 데이터 탐색, 시각화, 노이즈 제거, 학습 효율 향상을 위한 필수 전처리 기법이다. PCA는 빠르고 해석 가능한 선형 기법으로 지도 학습 전처리에 적합하고, t-SNE·UMAP은 군집 구조 시각화에 탁월하다. 공공 데이터 분석에서는 설명가능성을 위해 PCA 기반 전처리 후 모델링하는 방식이 권장되며, 주성분의 설명 분산 비율을 보고서에 명시해야 한다.

Ⅰ. 가. 이상 탐지 개념 및 이상치 유형
이상 탐지(Anomaly Detection)는 대다수 정상 데이터의 패턴에서 현저히 벗어난 데이터 포인트(이상치, Anomaly·Outlier)를 자동으로 식별하는 기법이다. 레이블이 없거나 이상치가 극히 드문 데이터 환경에서 비지도 또는 반지도 학습 방식으로 동작하며, 분류 문제와 구분된다.
이상치 유형정의예시
점 이상치
(Point Anomaly)		개별 데이터 포인트 자체가 정상 분포에서 크게 벗어난 경우평균 거래금액 5만원 대비 단일 거래 500만원 발생
맥락 이상치
(Contextual Anomaly)		특정 맥락(시간·공간)에서만 이상으로 간주되는 경우. 다른 맥락에서는 정상한여름 기온 30°C는 정상, 한겨울 기온 30°C는 이상
집단 이상치
(Collective Anomaly)		개별 포인트는 정상이지만, 연속된 패턴이 집단적으로 이상인 경우정상 심박수지만 수 시간 연속 일정한 패턴은 부정맥 의심
이상 탐지 vs 분류 문제의 차이
분류: 레이블된 훈련 데이터 필요. 정상·이상 모두 학습
이상 탐지: 이상 레이블 없거나 매우 희소 → 정상 패턴만 학습 후 이탈 탐지
이상치 비율 0.1~1% 수준 → 불균형 분류 문제로도 접근 가능
Ⅱ. 나. 이상 탐지 알고리즘 비교
기법원리이상 판정 기준장점단점적합 상황
Z-Score
(통계 기반)		각 값이 평균에서 표준편차의 몇 배인지 측정|z| = |x-μ|/σ > 3 이상치단순, 빠름, 해석 쉬움정규분포 가정. 다변량 이상 탐지 어려움단일 변수, 정규 분포 데이터
IQR
(비모수 통계)		4분위 범위 기반 이상 경계 설정x < Q1-1.5·IQR 또는 x > Q3+1.5·IQR분포 가정 없음, 강건다변량 미지원. 상자수염 그림과 동일비정규 분포, 단변량 탐색
LOF
(밀도 기반)		각 포인트의 지역 밀도를 k-최근접 이웃 밀도와 비교LOF 점수 >> 1 이상치 (주변보다 밀도 낮음)밀도 불균일 데이터 처리. 지역적 이상 탐지k 선택 민감. 고차원 성능 저하. O(n²)군집 크기·밀도 불균일 데이터
Isolation Forest
(트리 기반)		랜덤 분할로 데이터를 격리. 이상치는 적은 분할로 격리됨평균 경로 길이 짧을수록 이상 점수 높음빠름 O(n), 고차원 적합, 스케일 불필요지역적 이상 탐지 약함고차원, 대용량, 글로벌 이상 탐지
Autoencoder
(딥러닝 기반)		정상 데이터로 훈련된 오토인코더의 복원 오류(Reconstruction Error)가 크면 이상복원 오류 > 임계값 이상치비선형 복잡 패턴 학습. 시계열·이미지 이상 탐지학습 비용 높음. 임계값 설정 어려움이미지·시계열·로그 데이터 이상 탐지
LOF (Local Outlier Factor) 계산: 1. k-최근접 이웃(k-NN) 거리 계산 2. 도달 가능 거리(Reachability Distance): reach-dist_k(p, o) = max(k-dist(o), dist(p, o)) 3. 지역 도달 가능 밀도(LRD): lrd_k(p) = 1 / (Σ reach-dist_k(p,o) / k) 4. LOF 점수: LOF_k(p) = (Σ lrd_k(o)/lrd_k(p)) / k LOF ≈ 1: 정상, LOF >> 1: 이상치
