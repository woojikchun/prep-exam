M/M/1 핵심 수식 — 계산 문제 직결

λ=도착률, μ=서비스율, ρ=λ/μ (이용률). 안정 조건: ρ < 1 (반드시 확인!)
성능 지표수식의미
이용률ρ = λ/μ서버가 바쁜 비율
시스템 내 평균 고객 수 (L)L = ρ/(1-ρ)대기 + 서비스 중 고객 수
대기열 내 평균 고객 수 (Lq)Lq = ρ²/(1-ρ)서비스 대기 중인 고객 수
평균 체류 시간 (W)W = 1/(μ-λ)입장부터 퇴장까지 총 시간
평균 대기 시간 (Wq)Wq = ρ/(μ-λ)서비스 받기 전 대기 시간
리틀의 법칙 (범용)L = λW, Lq = λWq큐 모델·규율 무관하게 항상 성립

📐 기출 계산 예제 — λ=8/초, μ=10/초

이용률 ρ = 8/10= 0.8 (80%)
L = 0.8/(1-0.8)= 4명
Lq = 0.8²/(1-0.8)= 3.2명
W = 1/(10-8)= 0.5초
Wq = 0.8/(10-8)= 0.4초
리틀 법칙 검증: L = 8×0.5= 4명 ✓
λ=9(ρ=0.9)로 증가 시 L=9명, W=1초로 폭발적 증가. ρ가 1에 가까울수록 대기가 비선형 증가 — 이것이 용량 설계 시 ρ≤0.7을 권장하는 핵심 이유.
