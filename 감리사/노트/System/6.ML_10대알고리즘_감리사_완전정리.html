① 선형 회귀 (Linear Regression) ⭐

ALGORITHM #01 · 회귀 · 지도학습
핵심 원리 & 수식
▾
  • 목적: 입력 특성(X)의 선형 결합으로 연속형 출력(y) 예측
  • 핵심: 잔차(예측값-실제값)의 제곱합을 최소화하는 가중치 w와 절편 b를 찾음
  • 학습 방법: ①정규 방정식(Normal Equation, 소규모) ②경사 하강법(대규모)
ŷ = w₁x₁ + w₂x₂ + ... + wₙxₙ + b = Xw + b MSE = (1/n)Σ(yᵢ - ŷᵢ)² ← 최소화 목표 R² = 1 - SS_res/SS_tot ← 설명력 (1에 가까울수록 좋음)
  • 다중공선성(Multicollinearity): 독립 변수 간 높은 상관관계 → 계수 불안정 → VIF로 진단
  • Ridge(L2): 계수 축소·다중공선성 완화 / Lasso(L1): 일부 계수 0→특성 선택
  • ElasticNet: Ridge + Lasso 혼합
선형회귀=연속값예측·MSE최소화 / R²=설명력 / Ridge=L2·계수축소 / Lasso=L1·특성선택(0화)
📊

② 로지스틱 회귀 (Logistic Regression) ⭐

ALGORITHM #02 · 분류 · 지도학습 · "이름은 회귀지만 분류!"
핵심 원리 & 수식
▾
  • 목적: 이진 분류 — 클래스 1에 속할 확률을 0~1로 예측
  • 시그모이드: 선형 결합 결과를 0~1 확률로 변환
  • 결정 경계: P ≥ 0.5 → 클래스 1, P < 0.5 → 클래스 0
  • 손실 함수: Binary Cross-Entropy (Log Loss)
  • 다중 분류: One-vs-Rest(OvR) 또는 Softmax(Multinomial)
σ(z) = 1 / (1 + e-z) ← 시그모이드 (z = Xw + b) Log Loss = -[y·log(ŷ) + (1-y)·log(1-ŷ)] 오즈비(Odds Ratio) = P/(1-P) 로짓(Logit) = log(P/(1-P)) = Xw + b ← 선형 관계
로지스틱 회귀는 이름에 "회귀"가 있지만 분류 알고리즘! 확률을 출력하는 선형 결정 경계 분류기
로지스틱회귀=이름은회귀지만분류! / 시그모이드(0~1) / Binary Cross-Entropy / 다중분류=Softmax 또는 OvR
🌳

③ 결정 트리 (Decision Tree) ⭐

ALGORITHM #03 · 분류·회귀 · 지도학습 · 화이트박스
분할 기준 — 지니 vs 엔트로피 ⭐
▾
지니 불순도 G(t) = 1 - Σpᵢ² → 0에 가까울수록 순수 (모두 같은 클래스) → 최대값: 0.5 (이진 분류 시 균등 분포) 엔트로피 H(t) = -Σpᵢ·log₂(pᵢ) → 0에 가까울수록 순수 → 최대값: 1.0 (이진 균등 분포) 정보 이득 IG = H(부모) - Σ(|자식|/|부모|)·H(자식) → IG가 클수록 좋은 분할 기준
  • CART: 지니 불순도 사용 / 이진 분할 / sklearn 기본
  • ID3: 엔트로피·정보이득 / 범주형만 / 가지치기 없음
  • C4.5: ID3 개선 / 이득 비율(Gain Ratio) / 연속형 지원
과적합 방지 — 가지치기 (Pruning) ⭐
▾
  • 사전 가지치기(Pre-Pruning): 트리 생성 중 조기 중단 — max_depth·min_samples_split·min_samples_leaf
  • 사후 가지치기(Post-Pruning): 완성된 트리에서 불필요한 가지 제거 — 오류율 기반
  • 단점: 작은 데이터 변화에 민감(불안정) → 앙상블(랜덤 포레스트)로 극복
장점: 해석 용이(화이트박스) / 스케일링 불필요 / 비선형 패턴 / 혼합 데이터 처리
단점: 과적합 취약 / 데이터 변화에 불안정 / 선형 관계 비효율
결정트리: 지니(CART)·엔트로피(ID3·C4.5)로 분할 / 가지치기로 과적합 방지 / 화이트박스·스케일링불필요 / 단독사용시 과적합위험
⚔️

④ SVM (Support Vector Machine) ⭐

ALGORITHM #04 · 분류·회귀 · 지도학습 · 마진 최대화
핵심 원리 & 커널 트릭 ⭐
▾
  • 마진(Margin): 두 클래스를 구분하는 결정 경계와 가장 가까운 데이터 포인트(서포트 벡터) 간 거리 → 이를 최대화
  • 서포트 벡터: 결정 경계 결정에 실제로 관여하는 데이터 포인트만 (소수)
  • C 파라미터: 마진 vs 오분류 트레이드오프 / C↑=좁은마진·적은오분류(과적합) / C↓=넓은마진·허용오분류
  • 커널 트릭(Kernel Trick): 선형 분리 불가 데이터를 고차원으로 변환하여 선형 분리
커널수식특징
선형(Linear)K(x,z) = xᵀz선형 분리 가능 데이터·텍스트
다항식(Polynomial)K(x,z) = (xᵀz+c)ᵈ곡선 경계
RBF(가우시안) ⭐K(x,z) = exp(-γ||x-z||²)가장 많이 사용·임의 경계
시그모이드tanh(αxᵀz+c)신경망 유사
SVM은 특성 스케일링 필수! γ↑=복잡한 경계(과적합) / γ↓=단순한 경계(과소적합)
SVM=마진최대화·서포트벡터만 사용 / RBF커널=가장 많이 사용 / C·γ 하이퍼파라미터 / 스케일링 필수 / 대용량 느림
👥

⑤ KNN (K-Nearest Neighbors) ⭐

ALGORITHM #05 · 분류·회귀 · 지도학습 · 게으른 학습(Lazy Learning)
핵심 원리 & K 선택 ⭐
▾
  • 원리: 새 데이터와 가장 가까운 K개 이웃의 클래스 다수결(분류) 또는 평균(회귀)으로 예측
  • 게으른 학습: 별도 모델 학습 없이 전체 훈련 데이터를 메모리에 저장 → 학습 없음·예측 느림 O(n)
  • K=1: 과적합 위험 / K=n: 과소적합 / 홀수 권장(동점 방지)
  • 거리 척도: 유클리드(기본)·맨해튼·민코프스키
  • 차원의 저주(Curse of Dimensionality): 고차원에서 모든 점이 멀어져 성능 저하
  • 스케일링 필수: 거리 기반이므로 표준화·정규화 필요
유클리드 거리 = √Σ(xᵢ-zᵢ)² 맨해튼 거리 = Σ|xᵢ-zᵢ|
KNN=K이웃다수결·게으른학습(모델없음) / K홀수권장(동점방지) / 스케일링필수 / 예측O(n)·느림 / 고차원=차원의저주
🧮

⑥ 나이브 베이즈 (Naïve Bayes) ⭐

ALGORITHM #06 · 분류 · 지도학습 · 확률 기반 · 텍스트 분류 최강
핵심 원리 & 베이즈 정리 ⭐
▾
  • 핵심 가정: 모든 특성이 조건부 독립(Naïve) — 현실에서 위배되지만 실용적으로 잘 동작
  • 원리: 주어진 데이터에서 각 클래스에 속할 사후 확률 계산 → 최대 확률 클래스 선택
P(y|X) ∝ P(y) · Π P(xᵢ|y) ← Naïve 가정으로 곱 형태 사후확률 = 사전확률 × 우도(각 특성 조건부 확률 곱)
종류특성 분포 가정적합 데이터
가우시안 NB정규 분포연속형 특성
다항식 NB ⭐다항 분포단어 빈도·텍스트 분류
베르누이 NB베르누이 분포이진 특성 (단어 출현 여부)
  • 라플라스 스무딩: 훈련 데이터에 없던 단어 → 확률 0 방지 (분자에 1 추가)
  • 장점: 매우 빠른 학습·예측 / 소규모 데이터 효과적 / 텍스트 분류 탁월
나이브베이즈=베이즈정리+조건부독립가정 / 다항식NB=텍스트분류·단어빈도 / 라플라스스무딩=0확률방지 / 빠름·소규모탁월

⑩ 인공 신경망 (ANN / MLP) ⭐

ALGORITHM #10 · 분류·회귀·생성 · 지도·비지도 · 범용 함수 근사
핵심 구조 & 역전파 ⭐
▾
  • 퍼셉트론(Perceptron): 입력의 가중합에 활성화 함수 적용 → 뉴런 모방
  • 다층 퍼셉트론(MLP): 입력층 + 은닉층(1개↑) + 출력층 — 비선형 관계 학습
  • 활성화 함수: ReLU(은닉층 기본) / Sigmoid(이진 출력) / Softmax(다중 출력) / GELU(Transformer)
  • 순전파→손실계산→역전파→가중치 업데이트 반복
  • 역전파(Backpropagation): 연쇄 법칙으로 기울기 계산 후 경사 하강으로 가중치 업데이트
  • 최적화: SGD→Momentum→Adam(표준)→AdamW(Transformer)
  • 과적합 방지: Dropout / 배치 정규화 / 조기 종료 / L2 정규화
활성화 함수범위주요 용도특이사항
ReLU[0, ∞)은닉층 기본기울기 소실 완화 / Dead Neuron
Sigmoid(0, 1)이진 분류 출력기울기 소실 문제
Softmax(0,1) 합=1다중 분류 출력확률 분포 출력
Tanh(-1, 1)RNN0 중심·기울기 소실 여전히
ANN=역전파+경사하강 / ReLU=은닉층기본·기울기소실완화 / Softmax=다중분류출력 / Adam=표준 옵티마이저 / Dropout=과적합방지

⑧ 랜덤 포레스트 (Random Forest) ⭐

ALGORITHM #08 · 분류·회귀 · 지도학습 · 배깅 대표
배깅 + 무작위 특성 선택 ⭐
▾
  • 두 가지 무작위성: ①부트스트랩 샘플링(행 무작위) ②각 분할 시 특성 부분집합(열 무작위)
  • 앙상블: 여러 결정 트리의 예측을 다수결(분류) 또는 평균(회귀)으로 최종 예측
  • OOB(Out-of-Bag): 부트스트랩에 포함 안 된 샘플로 자체 검증 가능
  • 특성 중요도: 각 특성의 불순도 감소 기여도 → 특성 선택에 활용
  • 장점: 과적합 강건 / 높은 정확도 / 결측값 처리 / 특성 중요도 제공
  • 단점: 해석 어려움(블랙박스) / 메모리·시간 많음
