3D 프린터 (적층 제조, AM) 정의

3D 프린터: 디지털 3D 모델 데이터를 기반으로 재료를 층층이 쌓아 올려(Additive Manufacturing) 실물 입체 물체를 제작하는 기술 — ISO/ASTM 52900 표준

기존 제조 vs 3D 프린팅 비교

구분기존(절삭·사출)3D 프린팅
제조 방식절삭·제거(Subtractive)적층·추가(Additive)
재료 낭비많음최소화
금형필요불필요
소량 제작비효율효율적
복잡 형상어려움자유롭게 가능
생산 속도대량 생산 빠름대량 생산 느림

3D 프린팅 핵심 특징

  • 레이어-바이-레이어(Layer-by-Layer): 2D 단면을 순차적으로 적층하여 3D 형상 생성
  • 디지털 제조: 3D 모델→슬라이싱→출력 완전 디지털 워크플로우
  • 맞춤 제작(Customization): 개인별 맞춤 의족·임플란트 등
  • 래피드 프로토타이핑(RP): 빠른 시제품 제작·검증
  • 분산 제조: 설계 파일 전송 → 현지 출력 (물류 혁신)
  • 복잡 내부 구조: 격자(Lattice)·내부 통로 등 기존 불가 형상
3D 프린터 = 적층 제조(AM) / 금형 불필요·재료 낭비 최소화·복잡 형상 가능 / 기존 제조(절삭·제거) vs 3D(적층·추가)

주요 3D 프린팅 방식 상세 비교 ⭐

FDM (Fused Deposition Modeling) — 열용해 적층 ⭐가장 보편적
▾
  • 원리: 열가소성 플라스틱 필라멘트를 노즐에서 가열·용융하여 압출하며 한 층씩 쌓음
  • 장점: 저렴한 장비·재료 / 다양한 소재 / 사용 쉬움 / 대형 출력 가능
  • 단점: 표면 거칠음(레이어 라인) / 서포트 필요 / 이방성(레이어 방향 강도 약함)
  • 소재: PLA·ABS·PETG·TPU·나일론·탄소섬유 복합 필라멘트
  • 정밀도: 중간 (0.1~0.4mm 레이어)
  • 용도: 가정용·교육·시제품·기능성 부품·산업용 지그·픽스처
SLA (Stereolithography) — 광경화 방식 ⭐고정밀
▾
  • 원리: 액체 광경화 레진(수지) 조에 자외선(UV) 레이저를 조사하여 레이어별로 경화
  • DLP(Digital Light Processing): 레이저 대신 프로젝터(DMD 칩)로 한 레이어 전체를 한 번에 경화 → SLA보다 빠름
  • MSLA(Masked SLA): LCD 스크린으로 광원 마스킹 → 저가 레진 프린터 표준
  • 장점: 매우 높은 정밀도·표면 품질 / 세밀한 디테일
  • 단점: 후처리 필요(세척·후경화) / 레진 취성(깨짐) / 광 안전 주의 / 비쌈
  • 용도: 치과·의료 모델 / 보석·예술품 / 정밀 부품
SLS (Selective Laser Sintering) — 선택적 레이저 소결 ⭐서포트 불필요
▾
  • 원리: 분말 베드에 레이저로 분말을 선택적으로 소결(고온 결합)하여 형상 생성 — 미소결 분말이 서포트 역할
  • 핵심 특징: 서포트 구조물 불필요 — 주변 분말이 지지 → 복잡한 내부 형상·언더컷 자유롭게 제작
  • 소재: 나일론(PA12·PA11)·폴리프로필렌·TPU 분말
  • 장점: 서포트 없음 / 기계적 강도 우수 / 기능성 부품
  • 단점: 장비 고가 / 분말 회수·관리 필요 / 표면 다소 거침
  • 용도: 복잡한 기능성 부품 / 소량 생산 / 의료 보조기
SLM / DMLS — 금속 3D 프린팅 ⭐금속출력
▾
  • SLM (Selective Laser Melting): 금속 분말을 완전 용융하여 결합 — 100% 밀도 달성 가능
  • DMLS (Direct Metal Laser Sintering): 금속 분말을 소결하여 결합 (합금 소재에 주로 사용)
  • EBM (Electron Beam Melting): 전자빔으로 금속 분말 용융 — 진공 챔버 / 티타늄 특화
  • 소재: 티타늄·스테인리스·알루미늄·코발트크롬·인코넬
  • 장점: 복잡한 금속 부품 / 고강도 / 기존 제조 불가 형상
  • 단점: 매우 고가 / 후처리(열처리·표면처리) / 잔류응력
  • 용도: 항공우주(엔진 부품) / 의료(임플란트·보철) / 방산
방식정밀도속도비용서포트주요 용도
FDM중간중간저필요시제품·교육·기능부품
SLA/DLP최고빠름(DLP)중간필요치과·정밀 모델
SLS높음중간고불필요복잡 기능부품
SLM/DMLS높음느림매우 고필요금속·항공·의료

FDM=열용해·저가·보편 / SLA=UV레이저·고정밀·레진 / SLS=레이저소결·서포트불필요 / SLM=금속완전용융·항공의료

3D 프린팅 장단점 & 이슈

✅ 주요 장점

  • 래피드 프로토타이핑: 설계→시제품 시간 대폭 단축
  • 맞춤화(Customization): 개인별 맞춤 제품 경제적 생산
  • 복잡 형상: 기존 제조 불가능한 내부 구조·격자 구조
  • 재료 효율: 필요한 부분에만 재료 사용 → 낭비 최소화
  • 분산 제조: 설계 파일만 전송 → 현지 생산 → 물류 절감
  • 소량 제작: 금형 비용 없이 소량·다품종 경제적

⚠️ 주요 단점 & 과제

  • 출력 속도: 대량 생산에서 기존 방식보다 느림
  • 표면 품질: 레이어 라인 등 후처리 필요
  • 소재 한계: 모든 소재 지원 불가 / 특성 편차
  • 지적 재산권: 설계 파일 복제·불법 복제 문제
  • 규제·안전: 총기·의료기기 무허가 출력 위험
  • 이방성: 레이어 방향 강도 약함 (FDM 특히)
