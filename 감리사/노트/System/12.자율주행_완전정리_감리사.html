자율주행 핵심 용어

용어설명
ODD (Operational Design Domain)자율주행 시스템이 정상 작동하도록 설계된 조건 범위 — 도로 유형·날씨·속도·지리적 영역
OEDR (Object and Event Detection & Response)물체·사건 감지 및 대응 — 자율주행의 핵심 기능 (인지+판단+제어)
DDT (Dynamic Driving Task)실시간 운전 작업 전체 (조향·가감속·경로 계획·위험 대응)
MRC (Minimal Risk Condition)시스템 한계 도달 시 안전하게 차량을 정지·대피시키는 최소 위험 상태
AEB (Automatic Emergency Braking)전방 충돌 위험 감지 시 자동 긴급 제동 (L0에도 탑재)
ADAS (Advanced Driver Assistance System)L1~L2 수준의 첨단 운전자 보조 시스템 총칭
Functional Safety (ISO 26262)자동차 기능 안전 표준 — ASIL(A~D) 등급으로 안전 요구사항 분류
ODD=작동설계영역 / DDT=실시간운전작업 / MRC=최소위험상태 / ADAS=L1~L2보조시스템 / ISO26262=기능안전표준

센서 퓨전 (Sensor Fusion) ⭐ — 다중 센서 통합
▾
  • 정의: 여러 센서의 데이터를 결합하여 단일 센서의 한계를 보완하고 더 정확한 환경 인식을 달성
  • 초기 퓨전(Early Fusion): 원시 데이터 수준에서 결합 → 최대 정보 활용 / 계산 복잡
  • 후기 퓨전(Late Fusion): 각 센서의 결과(인식 후)를 결합 → 구현 단순 / 정보 손실
  • 중간 퓨전(Middle Fusion): 특성 추출 단계에서 결합 → 균형적
  • 칼만 필터(Kalman Filter): 센서 퓨전의 대표 알고리즘 — 노이즈 있는 측정값에서 최적 상태 추정
  • Tesla의 접근: 카메라만 사용(Camera-Only) — LiDAR 불필요 주장 (비용 절감, AI 의존)
  • Waymo의 접근: LiDAR+카메라+레이더 모두 사용 — 높은 비용, 높은 안전성
LiDAR=3D포인트클라우드·야간강·악천후취약 / 레이더=악천후강·속도측정(도플러) / 카메라=색상·표지판 / 센서퓨전=단점보완

인지 (Perception) — AI 기반 환경 인식
▾
  • 객체 탐지(Object Detection): YOLO·SSD·Faster R-CNN — 차량·보행자·표지판 실시간 감지
  • 의미론적 분할(Semantic Segmentation): 픽셀별 클래스 분류 — 도로·하늘·건물 구분
  • 3D 객체 탐지: LiDAR 포인트 클라우드 기반 — PointNet·VoxelNet
  • 차선 인식: OpenCV·딥러닝 — 차선 경계 추출 및 추적
  • 교통 신호 인식: CNN 기반 색상·형태 분류 (빨강·노랑·초록)
  • 보행자 예측: 보행자의 미래 이동 경로 예측 — LSTM·Transformer 기반
  • 점유 격자(Occupancy Grid): 공간을 격자로 나누어 각 셀의 점유 확률 표현
판단·계획 (Planning) — 3단계 계층 구조 ⭐
▾
계획 단계역할시간 범위
미션 계획 (Mission Planning)출발지→목적지 전체 경로 계획 (A*·Dijkstra·HD맵)수십 분~시간
행동 계획 (Behavioral Planning)차선 변경·추월·교차로 통과 등 전술적 행동 결정 — 상태 기계(State Machine)·MCTS수 초~수십 초
동작 계획 (Motion Planning)충돌 없이 부드러운 실제 궤적 생성 — Frenet Frame·Polynomial·RRT·MPC수백 ms
  • 비용 함수(Cost Function): 안전·효율·편안함·규정 준수를 가중치로 최적 경로 선택
  • 강화학습 적용: 복잡한 시나리오(교차로·합류)에서 학습 기반 행동 결정
위치 추정 (Localization) — SLAM & HD맵 활용
▾
  • SLAM (Simultaneous Localization And Mapping): 지도 작성과 위치 추정을 동시 수행 — 미지 환경에서 활용
  • NDT (Normal Distribution Transform): LiDAR 스캔을 HD맵과 매칭하여 정밀 위치 추정
  • 파티클 필터(Particle Filter): 가능한 위치 후보를 확률 분포로 표현·갱신
  • 칼만 필터(Kalman Filter): GPS·IMU·LiDAR 측정값을 융합하여 최적 위치 추정
  • 정밀도: GPS 단독=수 m 오차 → 센서 퓨전+HD맵 매칭=수 cm 오차
인지(센서→물체인식) → 위치추정(HD맵+GPS+LiDAR→cm급) → 계획(미션·행동·동작 3단계) → 제어(조향·가속·제동)

V2X (Vehicle-to-Everything) 통신 ⭐

V2X: 차량이 다른 차량·인프라·보행자·네트워크와 통신하여 센서만으로는 불가능한 정보를 획득하는 협력 지능형 교통 시스템
유형Full Name통신 대상주요 용도
V2V ⭐Vehicle-to-Vehicle다른 차량충돌 경고·전방 교통 정보·협력 주행(군집주행·Platooning)
V2I ⭐Vehicle-to-Infrastructure도로 인프라신호등 정보·도로 상태·과속 카메라·공사 구간 알림
V2PVehicle-to-Pedestrian보행자·자전거스마트폰 기반 보행자 위치 공유·충돌 경고
V2NVehicle-to-Network클라우드·서버HD맵 실시간 업데이트·원격 모니터링·SW 업데이트(OTA)
V2GVehicle-to-Grid전력망전기차 배터리로 전력망 균형 조절·양방향 충방전

HD맵 (High Definition Map) ⭐

HD맵: cm급 정밀도의 3D 고정밀 디지털 지도 — 자율주행 차량의 위치 추정·경로 계획의 기반

HD맵 구성 레이어

  • 도로 기하(Road Geometry): 차선 경계·곡률·경사·폭 (cm 정밀도)
  • 의미론적 레이어: 신호등·표지판·횡단보도·제한속도 위치
  • 동적 레이어: 실시간 교통·공사·사고 정보 (V2N 업데이트)
  • 3D 지형: 건물·고도 정보·터널·교량 구조

HD맵 활용 & 표준

  • 활용: LiDAR 맵 매칭으로 cm급 위치 추정 / 사전 경로 계획 / 보이지 않는 전방 정보
  • NDS (Navigation Data Standard): 차량 항법 데이터 표준
  • OpenDRIVE: 도로 네트워크 기술 XML 표준
  • OpenLABEL: AI 학습용 주석 표준 (ASAM)
  • 과제: 실시간 업데이트·대용량 데이터·개인정보 보호
HD맵=cm급 정밀 3D 디지털지도 / 4레이어: 도로기하·의미론·동적·3D지형 / LiDAR 매칭으로 cm급 위치추정

군집 주행 (Platooning) & 협력 주행

기술설명효과
군집 주행 (Platooning)선두 차량을 후속 차량들이 V2V 통신·자동 추종 — 최소 차간 거리 유지연료 절감(공기 저항↓)·교통 흐름 개선·물류 효율↑
CACC (협력 ACC)기존 ACC에 V2V 정보 추가 — 선두 차량의 가감속 즉시 반영반응 시간 단축·안전 거리 최소화
협력 교차로교차로에서 차량들이 V2I로 진입 시간 협상 — 신호 없이 통과교차로 처리량↑·정지 시간↓
OTA (Over-The-Air)무선으로 자율주행 소프트웨어 업데이트 — Tesla 등 상용화보안 패치·기능 추가·리콜 대체

  • UN Regulation 155: 자동차 사이버 보안 규정 — 유럽·일본·한국 등 적용 의무화
  • PKI (Public Key Infrastructure): V2X 메시지 서명·인증으로 위·변조 방지
  • 보안 부팅(Secure Boot): 펌웨어 서명 검증 → 무결성 보장
  • HSM (Hardware Security Module): 차량 내 암호화 키 안전 저장
