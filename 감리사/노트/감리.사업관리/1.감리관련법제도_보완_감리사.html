방식 1: 투입 인력 기준 ⭐⭐

감리에 투입되는 감리원의 인원 수 × 투입 기간(일수) × 일일 단가로 산정

감리 대가 = Σ(감리원 등급별 투입 일수 × 일일 단가) + 직접 경비(교통·숙박·인쇄 등) + 간접 비용(일반 관리비 등)
  • 적용: 복잡하거나 규모가 큰 사업
  • 장점: 실제 투입 노력 반영 정확
  • 단점: 사전 예측 어려움

방식 2: 사업비 요율 기준 ⭐⭐

사업비에 감리 요율(%)을 곱하여 산정 — 간편하고 예측 가능

감리 대가 = 사업비 × 감리 요율(%) 감리 요율 범위: • 5억 ~ 10억 : 2.0% ~ 3.5% • 10억 ~ 50억 : 1.5% ~ 3.0% • 50억 ~ 100억 : 1.0% ~ 2.5% • 100억 이상 : 0.8% ~ 2.0%
  • 적용: 일반적 사업
  • 장점: 계산 간편·예산 수립 용이
  • 단점: 사업 복잡성 미반영 가능

감리 대가 예산 계상 원칙 ⭐

  • 예산 분리 계상 ⭐: 감리 비용은 사업비와 별도로 계상
  • 의무 예산 확보: 의무 감리 대상 사업은 반드시 감리 예산 편성
  • 계약 방식: 일반 경쟁 입찰 또는 수의 계약
  • 최저가 배제: 덤핑 방지 — 적정 대가 보장
  • 부가세 별도: 감리 대가에 부가가치세 10% 별도

감리 대가 계산 실습 & 사례 ⭐⭐

사례 1: 사업비 요율 기준 계산 ⭐⭐
▾

📋 문제 조건

사업비 15억 원인 공공기관의 행정 정보시스템 신규 구축 사업. 감리 요율 2.5% 적용 시 감리 대가는?

// 1단계: 의무 감리 대상 확인 사업비 15억 > 5억 → ✅ 의무 감리 대상 // 2단계: 감리 횟수 결정 10억~20억 → 2회 이상 감리 필요 (설계·시험 단계) // 3단계: 감리 대가 계산 (요율 기준) 감리 대가 = 사업비 × 감리 요율 = 15억 × 2.5% = 3,750만 원 // 4단계: 부가세 포함 최종 금액 = 3,750만 + 375만(부가세10%) = 4,125만 원 // 의무감리 대상이므로 사업 예산에 감리비 3,750만~4,125만 원 별도 계상
사례 2: 투입 인력 기준 계산 ⭐⭐
▾

📋 문제 조건

사업비 30억 원인 정보시스템 사업에 대한 설계 단계 감리. 선임 감리원 1명 10일, 감리원 2명 각 10일 투입. 선임 단가 80만원/일, 감리원 단가 60만원/일. 간접 비용율 12%, 이윤율 8%

// 1단계: 직접 인건비 선임 감리원: 1명 × 10일 × 80만원 = 800만원 감리원(2명): 2명 × 10일 × 60만원 = 1,200만원 직접 인건비 합계 = 2,000만원 // 2단계: 직접 경비 (가정: 출장비 등 100만원) 직접 경비 = 100만원 // 3단계: 간접 비용 간접 비용 = 2,000만원 × 12% = 240만원 // 4단계: 이윤 이윤 = (2,000 + 100 + 240)만원 × 8% = 2,340만원 × 8% = 약 187만원 // 5단계: 감리 대가 합계 감리 대가 = 2,000 + 100 + 240 + 187 = 2,527만원 부가세 포함 = 2,527 × 1.1 = 약 2,780만원 // 참고: 사업비 요율 기준 = 30억 × 2% = 6,000만원 (차이 발생 가능)
